# AI Dashboard

## Project Overview

This project is an interactive AI Dashboard built using the Iris Dataset.

## Features

* Dataset Overview
* Data Cleaning
* Key Performance Indicators (KPIs)
* Interactive Filter
* 5 Data Visualizations

## Technologies Used

* Python
* Pandas
* Plotly
* Streamlit

## Dataset

Iris Dataset

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the application

streamlit run app.py

## Output

The dashboard provides insights into the Iris dataset through interactive charts and filters.
